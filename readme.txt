This simple and small plugin removes the /blog/-Slug from your WordPress posts in the main blog of your MultiSite installation

== Installation ==

1. Upload the folder 'remove-blog-slug' to the plugin folder
2. Switch to blog 1 and activate this plugin
3. Head to Options -> Permalinks and refresh the permalinks
